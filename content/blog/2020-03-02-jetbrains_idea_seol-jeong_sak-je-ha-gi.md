Summary: JetBrains를 처음 설치한 것처럼 다시 설치해보자...

1. Tool box 로 설치한 기본 실행기 삭제
2. `~/Library/ApplicationSupport` 에 있는 플러그인 파일 삭제
3. `~/Library/Preferences/com.jetbrains.*` 에 있는 설정 파일 삭제
